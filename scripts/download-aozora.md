# 青空文庫テキスト取得方法

## 現在の実装

現在は、手動で青空文庫からテキストをコピー&ペーストして、ローカルファイルとして保存しています。

## 変換作業について

青空文庫のテキストファイルには以下の特徴があります：

1. **文字コード**: Shift_JISでエンコードされている
2. **ルビ記法**: 《ルビ》や｜《ルビ》の形式でルビが記述されている
3. **注釈**: ［＃注1］などの形式で注釈が記述されている
4. **改ページ指示**: ［＃改ページ］などの指示が含まれている

これらを処理する必要があるため、単純なコピー&ペーストでは不十分です。

## より効率的な方法

### 方法1: 青空文庫の公式サイトからダウンロード

1. 青空文庫の公式サイト（https://www.aozora.gr.jp/）にアクセス
2. 作品を検索
3. 作品ページから「テキストファイル（.txt）」をダウンロード
4. ダウンロードしたファイルを `apps/hosting/src/data/` に配置
5. ルビや注釈を処理するスクリプトで変換

### 方法2: スクリプトで自動処理（推奨）

`scripts/download-aozora.js` スクリプトを使用して、青空文庫のテキストを自動的に処理できます。

ただし、青空文庫のURL構造が作品によって異なるため、各作品の正確なURLを確認する必要があります。

### 方法3: 手動で全文をコピー&ペースト

最も確実な方法ですが、時間がかかります。

## 推奨される対応

現在の実装では、各作品の冒頭部分を提供していますが、全文を提供するには：

1. 青空文庫の公式サイトから各作品の全文テキストをダウンロード
2. ルビや注釈を処理するスクリプトで変換
3. `apps/hosting/src/data/` に配置

または、より長い抜粋を手動で追加することも可能です。
