# 無料で著作権侵害なく使用できる文章ソース

速読アプリのテキストソースとして使用できる、無料で著作権侵害の心配がない文章のソースをまとめます。

## 1. パブリックドメインの文学作品

### 青空文庫（既に使用中）
- **URL**: https://www.aozora.gr.jp/
- **説明**: 日本の文学作品を中心に、著作権が消滅した作品や著作者が公開を許可した作品を無料で提供
- **特徴**:
  - 著作者の死後70年が経過した作品
  - 著作者が公開を許可した作品
  - 主に明治・大正・昭和初期の文学作品
- **代表的作品**:
  - 夏目漱石「こころ」「吾輩は猫である」
  - 芥川龍之介「羅生門」「蜘蛛の糸」
  - 宮沢賢治「銀河鉄道の夜」
  - 太宰治「人間失格」「走れメロス」
- **注意点**: テキストファイルはShift_JISでエンコードされており、ルビ記法や注釈が含まれるため、処理が必要

### Project Gutenberg
- **URL**: https://www.gutenberg.org/
- **説明**: 海外のパブリックドメイン作品を多数収録
- **特徴**:
  - 主に英語の古典文学
  - 日本語作品も一部収録されている
  - 複数のフォーマット（TXT、HTML、EPUBなど）で提供
- **注意点**: 日本語作品は限定的

## 2. 公的機関が作成した文書（著作権が発生しない）

### 法律・法令
- **憲法、法律、政令、省令、告示、通達など**
- **特徴**: 著作権の対象外として自由に利用可能
- **例**:
  - 日本国憲法
  - 各種法律の条文
  - 内閣府や各省庁の告示・通達

### 裁判所の判決文
- **最高裁判所**: https://www.courts.go.jp/
- **特徴**: 判決文は著作権の対象外
- **注意点**: 判例集などに編集が加えられたものは編集著作物として著作権が発生する場合がある

### 政府統計・白書
- **総務省統計局**: https://www.stat.go.jp/
- **内閣府**: https://www.cao.go.jp/
- **特徴**: 政府が作成した統計資料や白書は、一般的に著作権の対象外
- **例**:
  - 国勢調査報告書
  - 各種白書（経済財政白書、防衛白書など）
  - 統計データの説明文

### 国会議事録
- **国会会議録検索システム**: https://kokkai.ndl.go.jp/
- **特徴**: 国会の議事録は著作権の対象外
- **注意点**: 議員の発言内容自体には著作権が発生する可能性があるため、利用時は注意が必要

## 3. クリエイティブ・コモンズ（CC）ライセンス

### CC0（パブリックドメイン相当）
- **説明**: 著作権者が著作権を放棄し、パブリックドメインと同等の状態にした作品
- **利用条件**: 制限なし（商用・非商用問わず自由に利用可能）

### CC BY（表示）
- **説明**: 出典を明記すれば自由に利用可能
- **利用条件**: 出典の明記が必要

### CC BY-SA（表示-継承）
- **説明**: 出典を明記し、同じライセンスで公開すれば自由に利用可能
- **利用条件**: 出典の明記と、同じライセンスでの公開が必要

### CCライセンス作品の検索
- **Wikimedia Commons**: https://commons.wikimedia.org/
- **Internet Archive**: https://archive.org/
- **Flickr**: https://www.flickr.com/creativecommons/

## 4. その他の無料テキストソース

### Wikipedia
- **URL**: https://ja.wikipedia.org/
- **ライセンス**: CC BY-SA 3.0
- **特徴**: 記事の本文は出典を明記すれば利用可能
- **注意点**: 画像や一部の引用には別のライセンスが適用される場合がある

### 古い新聞記事
- **国立国会図書館デジタルコレクション**: https://dl.ndl.go.jp/
- **特徴**: 著作権が消滅した古い新聞記事を閲覧可能
- **著作権について**:
  - **70年以上前の新聞記事**: 著作権の保護期間が終了しているため、理論上は自由に使用可能
  - **法人著作物としての保護期間**: 新聞記事は通常、新聞社（法人）が著作権を保有しており、公表後70年間保護される
  - **2024年時点**: 1954年以前に発行された新聞記事は、著作権の保護期間が終了している可能性が高い
- **注意点**:
  - **利用規約の確認**: 国立国会図書館デジタルコレクションの利用規約を必ず確認する必要がある
  - **商用利用の制限**: デジタルコレクションの利用規約では、商用利用が制限されている場合がある
  - **写真・図版の著作権**: 記事本文だけでなく、写真や図版には別の著作権が存在する可能性がある
  - **編集著作物**: 新聞全体が編集著作物として保護されている場合がある
  - **記者の著作権**: 記事の内容によっては、記者個人の著作権が関係する場合がある
- **「事実の伝達にすぎない雑報」**: 死亡記事、交通事故、人事異動などの簡潔な事実報道は著作物に該当しない（著作権法第10条第2項）
- **推奨事項**: 
  - 古い新聞記事を使用する場合は、著作権の保護期間を確認する
  - 国立国会図書館の利用規約を遵守する
  - 商用利用の場合は、特に注意が必要
  - 可能であれば、新聞社に確認を取る

### 学術論文（オープンアクセス）
- **J-STAGE**: https://www.jstage.jst.go.jp/
- **arXiv**: https://arxiv.org/
- **特徴**: オープンアクセスの論文は自由に利用可能
- **注意点**: 各論文のライセンスを確認する必要がある

## 5. 注意事項

### 著作権保護期間の確認
- **日本**: 著作者の死後70年（2024年時点では、1954年以前に死亡した著作者の作品）
- **注意**: 戦時加算により、実際の保護期間が延長されている場合がある

### 二次的著作物への注意
- パブリックドメインの作品でも、翻訳や編集が加えられた二次的著作物には新たな著作権が発生する場合がある
- 青空文庫のテキストファイルは、入力・校正作業が加えられているため、その部分に著作権が発生する可能性がある（ただし、青空文庫は利用を許可している）

### その他の権利
- **商標権**: 作品名や登場人物名が商標登録されている場合がある
- **肖像権**: 写真やイラストに肖像権が関係する場合がある
- **パブリシティ権**: 著名人の名前や肖像の商業利用には制限がある場合がある

### 利用時の推奨事項
1. **出典の明記**: パブリックドメインの作品でも、出典を明記することで情報の信頼性を高める
2. **利用規約の確認**: 各サイトの利用規約を確認する
3. **ライセンスの確認**: CCライセンスの場合は、ライセンスの種類を確認し、条件を遵守する
4. **定期的な確認**: 著作権の状況は変化する可能性があるため、定期的に確認する

## 6. 推奨される利用方法

### 速読アプリでの利用に適したソース
1. **青空文庫**: 既に使用中。日本語の文学作品が豊富で、長文も多い
2. **政府統計・白書**: 実用的な文章で、速読練習に適している
3. **Wikipedia**: 様々なトピックの文章があり、長さも調整しやすい
4. **Project Gutenberg**: 英語の練習にも対応できる

### 実装の優先順位
1. **青空文庫の拡充**: 既に実装済み。より多くの作品を追加
2. **政府統計・白書**: 実用的な文章として追加
3. **Wikipedia API**: 動的に文章を取得できるようにする
4. **Project Gutenberg**: 英語の練習にも対応
