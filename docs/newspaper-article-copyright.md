# 新聞記事の著作権と利用可能性について

## 1. 基本的な著作権の扱い

### 新聞記事は著作物として保護される
- 新聞記事は、一般的に著作権法で保護されている「著作物」に該当します
- 記事を無断でコピー、配布、またはインターネット上で公開することは、著作権侵害となる可能性があります
- 著作権者は通常、新聞社（法人）が保有しています

### 例外：「事実の伝達にすぎない雑報」
- 著作権法第10条第2項では、「事実の伝達にすぎない雑報および時事の報道」は著作物に該当しないとされています
- これには以下のような記事が含まれます：
  - 死亡記事
  - 交通事故の簡潔な報道
  - 人事異動の告知
  - その他の簡潔な事実報道
- ただし、記者の創作性が反映された記事（分析記事、コラム、特集記事など）は著作物として保護されます

## 2. 著作権の保護期間

### 法人著作物としての保護期間
- 新聞記事は通常、新聞社（法人）が著作権を保有しています
- 法人著作物の場合、**公表後70年間**保護されます
- 2024年時点では、**1954年以前に発行された新聞記事**は、著作権の保護期間が終了している可能性が高い

### 計算方法
- 発行年 + 70年 = 保護期間の終了年
- 例：1945年に発行された記事 → 2015年に保護期間終了
- 例：1950年に発行された記事 → 2020年に保護期間終了

### 注意点
- 戦時加算により、実際の保護期間が延長されている場合がある
- 個別の記事によって保護期間が異なる可能性がある
- 正確な保護期間を確認するには、専門家への相談が推奨されます

## 3. 国立国会図書館デジタルコレクション

### 利用可能な記事
- 国立国会図書館デジタルコレクションには、古い新聞記事が含まれています
- 主に明治・大正・昭和初期の新聞記事が閲覧可能です

### 利用規約の確認が必要
- **利用規約を必ず確認する必要があります**
- デジタルコレクションの利用規約では、以下のような制限がある場合があります：
  - 商用利用の制限
  - 再配布の制限
  - 改変の制限
- 利用規約に違反すると、著作権侵害とは別に、利用規約違反となる可能性があります

### 商用利用について
- デジタルコレクションの利用規約では、商用利用が制限されている場合があります
- 速読アプリのような商用サービスで使用する場合は、特に注意が必要です
- 可能であれば、国立国会図書館に問い合わせることを推奨します

## 4. その他の注意点

### 写真・図版の著作権
- 新聞記事には、写真や図版が含まれる場合があります
- これらの写真や図版には、記事本文とは別の著作権が存在する可能性があります
- 写真の著作権は、撮影者の死後70年間保護されます

### 編集著作物としての保護
- 新聞全体が編集著作物として保護されている場合があります
- 個々の記事の著作権が消滅していても、新聞全体の編集には著作権が存在する可能性があります

### 記者の著作権
- 記事の内容によっては、記者個人の著作権が関係する場合があります
- 特に署名記事やコラムなどは、記者個人の著作物として扱われる場合があります

## 5. 利用時の推奨事項

### 確認すべきこと
1. **著作権の保護期間**: 記事の発行年を確認し、70年以上経過しているか確認
2. **利用規約**: 国立国会図書館デジタルコレクションの利用規約を確認
3. **商用利用の可否**: 商用利用が許可されているか確認
4. **写真・図版**: 写真や図版の著作権を確認

### 推奨される対応
1. **古い記事を選ぶ**: 1954年以前（できれば1940年以前）の記事を選ぶ
2. **利用規約を遵守**: 国立国会図書館の利用規約を必ず遵守する
3. **出典を明記**: 記事の出典を明記する
4. **専門家に相談**: 不明な点がある場合は、専門家に相談する
5. **新聞社に確認**: 可能であれば、該当する新聞社に確認を取る

## 6. 結論

### 自由に使用可能な記事
- **70年以上前の新聞記事**: 理論上は著作権の保護期間が終了しているため、自由に使用可能
- **「事実の伝達にすぎない雑報」**: 簡潔な事実報道は著作物に該当しない

### 注意が必要な記事
- **70年以内の記事**: 著作権の保護期間内のため、新聞社の許諾が必要
- **写真・図版**: 別の著作権が存在する可能性がある
- **編集著作物**: 新聞全体の編集には著作権が存在する可能性がある

### 速読アプリでの利用について
- 古い新聞記事（1954年以前）を使用する場合は、理論上は著作権の保護期間が終了している可能性が高い
- ただし、国立国会図書館デジタルコレクションの利用規約を必ず確認する必要がある
- 商用利用の場合は、特に注意が必要
- **推奨**: 青空文庫の文学作品の方が、著作権の扱いが明確で安全
