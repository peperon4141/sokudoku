# 最近の作品で使用可能な文章（70年経過していないがCCライセンスなどで使用可能）

## 1. 青空文庫の「著作者が公開を許可した作品」

### 特徴
- 青空文庫には、著作権が消滅した作品だけでなく、**著作者が公開を許可した作品**も含まれています
- これらの作品は、著作権が存続していても、青空文庫の利用規約に従って利用可能です
- 各作品の利用条件を確認する必要があります

### 確認方法
- 青空文庫の各作品ページで、著作権情報を確認できます
- 「著作者が公開を許可した作品」と明記されている場合があります

### 注意点
- 青空文庫の利用規約を遵守する必要があります
- 商用利用が許可されているか確認が必要です
- 各作品の利用条件が異なる場合があります

## 2. Wikipedia（CC BY-SA 3.0）

### 特徴
- **大部分の記事がCC BY-SA 3.0**で公開されています
- 2020年代の最新記事も含まれます
- 様々なトピックの記事があります

### 利用可能な記事の例
- **歴史**: 戦国時代、幕末、世界史など
- **科学**: 宇宙、生物、物理など
- **文化**: 伝統文化、芸術、音楽など
- **人物**: 歴史上の人物、科学者、芸術家など
- **地理**: 世界の国々、日本の都道府県など
- **技術**: AI、プログラミング、最新技術など

### 利用条件
- 出典を明記する（「この記事はWikipediaから引用しました（CC BY-SA 3.0）」など）
- 同じライセンス（CC BY-SA）で公開する必要がある
- 商用利用可能

### 取得方法
- Wikipedia APIを使用して記事を取得
- 特定のトピックを検索して記事を取得

## 3. オープンアクセスの学術論文（J-STAGE）

### 特徴
- **J-STAGE**には、日本語の学術論文が多数公開されています
- 多くが**CC BY**ライセンスで公開されています
- 2020年代の最新論文も含まれます

### 利用可能な分野
- **人文科学**: 文学、歴史、哲学など
- **社会科学**: 経済、政治、社会学など
- **自然科学**: 物理、化学、生物など
- **工学**: コンピュータ、機械、電気など

### 利用条件
- 各論文のライセンスを確認する必要がある
- 多くがCC BY（出典を明記すれば利用可能）
- 商用利用可能（CC BYの場合）

### 取得方法
- J-STAGE APIを使用して論文を取得
- 特定の分野やキーワードで検索

## 4. 個人がCCライセンスで公開している作品

### 特徴
- 個人のブログやウェブサイトで、CCライセンスを適用した小説や文章を公開している場合があります
- 多くが**CC BY**または**CC BY-SA**で公開されています

### 探し方
- ブログ検索サイトで「CC BY」「クリエイティブ・コモンズ」などで検索
- 個人のウェブサイトでライセンス情報を確認

### 注意点
- 各作品のライセンスを確認する必要がある
- 出典を明記する必要がある（CC BY、CC BY-SAの場合）

## 5. オープンソースプロジェクトのドキュメント

### 特徴
- オープンソースプロジェクトのドキュメントは、多くがCCライセンスで公開されています
- 技術的な文章が多いですが、説明文として使用可能です

### 例
- **プログラミング言語のドキュメント**: Python、JavaScript、Rubyなど
- **フレームワークのドキュメント**: React、Vue.js、Djangoなど
- **ツールのドキュメント**: Git、Docker、Kubernetesなど

### 利用条件
- 多くがCC BYまたはCC BY-SA
- 出典を明記する必要がある

## 6. 政府・公的機関の文書

### 特徴
- 政府や公的機関が作成した文書は、著作権が発生しない場合があります
- 2020年代の最新の文書も含まれます

### 例
- **法律・法令**: 最新の法律も含まれます
- **白書**: 経済財政白書、防衛白書、環境白書など（最新版も含む）
- **統計資料**: 国勢調査、各種統計（最新版も含む）
- **報告書**: 各種審議会の報告書など

### 注意点
- 公的機関が作成した文書でも、編集やデザインには著作権が発生する場合があります
- 利用規約を確認する必要があります

## 7. 実際の利用例

### Wikipedia記事の例

- 「ChatGPT」の記事（2020年代の技術）

### 学術論文の例

### 政府文書の例

- 最新の統計資料

## 8. 速読アプリでの実装方法

### Wikipedia APIの実装
```typescript
// Wikipedia APIを使用して記事を取得
const fetchWikipediaArticle = async (title: string) => {
  const response = await fetch(
    `https://ja.wikipedia.org/api/rest_v1/page/summary/${encodeURIComponent(title)}`
  )
  const data = await response.json()
  return data.extract // 記事の抜粋
}
```
