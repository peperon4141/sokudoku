# 70年経過していないが使用可能な文章

著作権の保護期間が70年経過していない（まだ著作権が有効な）作品でも、以下の条件を満たす場合、無料で合法的に使用できる文章が存在します。

## 1. クリエイティブ・コモンズ（CC）ライセンスの作品

### CC0（パブリックドメイン相当）
- **説明**: 著作者が著作権を放棄し、パブリックドメインと同等の状態にした作品
- **利用条件**: 制限なし（商用・非商用問わず自由に利用可能）
- **出典明記**: 不要（推奨はされる）
- **ソース**: 
  - Wikimedia Commons
  - Internet Archive
  - 各種オープンソースプロジェクト

### CC BY（表示）
- **説明**: 出典を明記すれば自由に利用可能
- **利用条件**: 出典の明記が必要
- **商用利用**: 可能
- **改変**: 可能
- **ソース**: 
  - Wikipedia（一部の記事）
  - 各種オープンアクセスの学術論文

### CC BY-SA（表示-継承）
- **説明**: 出典を明記し、同じライセンスで公開すれば自由に利用可能
- **利用条件**: 出典の明記と、同じライセンスでの公開が必要
- **商用利用**: 可能
- **改変**: 可能（同じライセンスで公開する必要がある）
- **ソース**: 
  - Wikipedia（大部分の記事）
  - 各種オープンソースプロジェクト

### CC BY-NC（表示-非営利）
- **説明**: 出典を明記し、非営利目的でのみ利用可能
- **利用条件**: 出典の明記と、非営利目的での利用が必要
- **商用利用**: 不可
- **改変**: 可能
- **注意**: 速読アプリが商用サービスの場合、このライセンスの作品は使用できない

## 2. 著作者が公開を許可した作品

### 青空文庫の「著作者が公開を許可した作品」
- **説明**: 青空文庫には、著作権が消滅した作品だけでなく、著作者が公開を許可した作品も含まれています
- **特徴**: 
  - 著作権は存続しているが、著作者が公開を許可している
  - 青空文庫の利用規約に従って利用可能
- **利用条件**: 
  - 青空文庫の利用規約を遵守する
  - 出典を明記する（推奨）
- **注意点**: 
  - 各作品の利用条件を確認する必要がある
  - 商用利用が許可されているか確認が必要

### 文化庁の「自由利用マーク」
- **説明**: 著作者が自身の作品に対して特定の利用を許可する意思表示
- **種類**: 
  - 教育利用のみ許可
  - 非営利目的での利用を許可
  - 商用利用も許可
- **利用条件**: マークの種類によって異なる
- **ソース**: 文化庁のウェブサイト

## 3. オープンアクセスの学術論文

### J-STAGE（オープンアクセス論文）
- **URL**: https://www.jstage.jst.go.jp/
- **説明**: 日本の学術論文を公開
- **ライセンス**: 論文によって異なる（CC BY、CC BY-NCなど）
- **利用条件**: 各論文のライセンスを確認する必要がある
- **特徴**: 
  - 日本語の学術論文が豊富
  - 様々な分野の論文がある
  - 実用的な知識が得られる

### arXiv
- **URL**: https://arxiv.org/
- **説明**: 主に物理学、数学、コンピュータサイエンスの論文
- **ライセンス**: 論文によって異なる
- **特徴**: 
  - 主に英語
  - 最新の研究成果

## 4. Wikipedia

### ライセンス
- **大部分の記事**: CC BY-SA 3.0
- **利用条件**: 
  - 出典を明記する
  - 同じライセンス（CC BY-SA）で公開する必要がある
- **商用利用**: 可能
- **改変**: 可能（同じライセンスで公開する必要がある）

### 特徴
- 様々なトピックの記事がある
- 長さも調整しやすい
- 実用的な知識が得られる
- 日本語の記事が豊富

### 注意点
- 画像や一部の引用には別のライセンスが適用される場合がある
- 記事の内容は常に更新されているため、特定のバージョンを固定する必要がある場合がある

## 5. 著作権が発生しない文章

### 「事実の伝達にすぎない雑報」
- **説明**: 著作権法第10条第2項により、著作物に該当しない
- **例**: 
  - 死亡記事
  - 交通事故の簡潔な報道
  - 人事異動の告知
  - その他の簡潔な事実報道
- **利用条件**: 制限なし
- **注意点**: 記者の創作性が反映された記事は著作物として保護される

### 短いフレーズやありふれた表現
- **説明**: 創作性が認められないものは著作物と見なされない
- **例**: 
  - 短いキャッチコピー
  - ありふれた表現
- **注意点**: 俳句や短歌のように短くても創作性が高いものは著作物と認められる

## 6. 著作権法の例外規定

### 引用
- **説明**: 報道、批評、研究などの目的で、適切な範囲内での引用は許可されている
- **条件**: 
  - 引用の目的が正当であること
  - 引用部分が明確に区別されていること
  - 出典を明記すること
  - 引用が主、被引用が従の関係であること
- **注意点**: 速読アプリで全文を使用する場合は、引用の範囲を超える可能性がある

## 7. 速読アプリでの利用に適したソース

### 最適なソース
1. **Wikipedia（CC BY-SA）**
   - 様々なトピックから選択できる
   - 長さも調整しやすい
   - 出典を明記すれば利用可能
   - 同じライセンスで公開する必要がある

2. **青空文庫の「著作者が公開を許可した作品」**
   - 文学作品が豊富
   - 青空文庫の利用規約を遵守すれば利用可能
   - 各作品の利用条件を確認する必要がある

3. **オープンアクセスの学術論文（CC BY）**
   - 実用的な知識が得られる
   - 出典を明記すれば利用可能
   - 各論文のライセンスを確認する必要がある

### 注意が必要なソース
1. **CC BY-NC（非営利）ライセンス**
   - 商用サービスの場合は使用できない

2. **引用**
   - 全文を使用する場合は、引用の範囲を超える可能性がある

## 8. 実装の推奨事項

### Wikipedia APIの実装
- Wikipedia APIを使用して記事を取得
- 出典を明記する
- ライセンス情報を表示する
- 同じライセンス（CC BY-SA）で公開する

### 青空文庫の拡充
- 「著作者が公開を許可した作品」を追加
- 各作品の利用条件を確認
- 利用規約を遵守する

### オープンアクセスの学術論文
- J-STAGE APIを使用して論文を取得
- 各論文のライセンスを確認
- 出典を明記する

## 9. まとめ

### 70年経過していないが使用可能な文章
1. **CC0（パブリックドメイン相当）**: 制限なし
2. **CC BY（表示）**: 出典を明記すれば利用可能
3. **CC BY-SA（表示-継承）**: 出典を明記し、同じライセンスで公開すれば利用可能
4. **青空文庫の「著作者が公開を許可した作品」**: 青空文庫の利用規約を遵守すれば利用可能
5. **Wikipedia（CC BY-SA）**: 出典を明記し、同じライセンスで公開すれば利用可能
6. **オープンアクセスの学術論文（CC BY）**: 出典を明記すれば利用可能
7. **「事実の伝達にすぎない雑報」**: 制限なし

### 注意点
- 各ライセンスの条件を必ず確認する
- 出典を明記する（多くのライセンスで必要）
- 商用利用が許可されているか確認する
- 改変が許可されているか確認する
- 同じライセンスで公開する必要があるか確認する
