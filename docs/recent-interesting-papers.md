## 3. 論文の取得方法

### J-STAGE API
- **URL**: https://www.jstage.jst.go.jp/
- **API**: 公式APIが提供されている場合がある
- **検索**: キーワード、著者名、発行年で検索可能

### arXiv API
- **URL**: https://arxiv.org/
- **API**: https://arxiv.org/help/api
- **検索**: キーワード、著者名、カテゴリで検索可能

## 4. 実装の推奨事項

### 論文リーディングモードの機能
1. **論文検索**: キーワード、著者名、発行年で検索
2. **論文表示**: 要旨、全文を表示
3. **速読トレーニング**: 既存の速読メソッドで論文を読む
4. **ブックマーク**: 気に入った論文を保存
5. **出典表示**: ライセンス情報と出典を表示

### 注意事項
- **私的利用**: 個人で読む分には問題なし
- **ライセンス確認**: 各論文のライセンスを確認
- **出典明記**: CC BY、CC BY-SAの場合は出典を明記
